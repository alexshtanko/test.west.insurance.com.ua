== Действующие шорткоды ==

Partners System использует для вывода общей партнерской ссылки пользователя шорткод

[ ref-url ]

Если есть желание вывести партнерскую ссылку на какую то конкретную страницу, то в контенте этой страницы можно вывести шорткод

[ ref-url type='tail' ]

Он сформирует ссылку перейдя по которой посетитель будет попадать на ту страницу с которой эта партнерская ссылка была взята.

Для вывода карты рефералов партнера следует использовать шорткод

[ partner-map ]

Статистику партнера можно вывести на выбранной странице с помощью шорткода

[ partner-stats ]

== Changelog ==

= UPD 1.12.0 =

добавлена возможность 
 - вручную устанавливать связи партнер-реферал
 - перезакреплять реферала за другим партнером
 - переназначать партнера для реферала

= UPD 1.11.0 =

добавлена возможность формировать партнеру промокод, вывод поля для указания промокода в форме регистрации при регистрации реферала.
добавлена возможность возврата рефералу на его внутренний баланс указанного процента от суммы первого заказа (по-умолчанию работает с WooCommerce и дополнение Commerce WP-Recall)

= UPD 1.8.0 =

добавлена возможность ограничивать количество прямых рефералов для одного партнера.


= UPD 15.08.2015 =

Добавлена поддержка плагина Woocommerce.

= UPD ver. 1.4.0 =

Добавлена возможность указания определенного пользователя на сайте в качестве партнера по-умолчанию либо назначать партнера случайно, если пользователь регистрируется не по партнерской ссылке.
Добавлена опция позволяющая при удалении пользователя с сайта передавать всех закрепленных за ним рефералов вышестоящему партнеру, чьим рефералом он сам является.
Добавлена опция вывода информации о вышестоящем партнере в личном кабинете реферала.
